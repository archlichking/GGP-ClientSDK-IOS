Detailed Description

The GreeAchievement interface encapsulates all of the Gree Platform's achievement functionality.

An achievement is a discrete accomplishment made by the user in your application (i.e. beating level 3.) Achievements and their details are pre-defined and created on the GREE developer website. It is your responsibility to unlock the achievement at a particular point in your application once the user has earned it.
