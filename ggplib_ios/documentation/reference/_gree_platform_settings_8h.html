Detailed Description

GreePlatform initialization settings.

These constants should be used to build the settings dictionary for [GreePlatform initializeWithProductKey...].

Enumeration Type Documentation

enum GreeLogLevel

Pre-defined logging thresholds, from least-verbose to most-verbose.

Enumerator:
GreeLogLevelPublic 

GreeLogLevelPublic will show only the most essential log messages.

GreeLogLevelWarn 

GreeLogLevelWarn will show public messages as well as those designed to warn you of dangerous behavior.

GreeLogLevelInfo 

GreeLogLevelInfo will show most any log message, no matter how verbose or purely informational.

NSString* const GreeSettingInterfaceOrientation

Specify the orientation that Gree should use.

Your application, if it supports multiple orientations, is expected to update GreePlatform via +[GreePlatform setInterfaceOrientation:] when orientation changes occur. All views (dashboard, popups, notifications) will use this. Values are expected to be [NSNumber numberWithInteger:(UIInterfaceOrientation)].

NSString* const GreeSettingGameCenterAchievementMapping

Define mappings for achievements from GameCenter to Gree.

When this setting is enabled GreePlatform will submit achievements to GameCenter at the same time they are unlocked via the GreeAchievement APIs.

Expected value is an NSDictionary where the keys are Gree achievement identifiers (NSString) and the objects are GameCenter acheivement identifiers (NSString).

NSString* const GreeSettingGameCenterLeaderboardMapping

Define mappings for leaderboards from GameCenter to Gree.

When this setting is enabled GreePlatform will submit scores to GameCenter at the same time they are submitted via the GreeScore APIs.

Expected value is an NSDictionary where the keys are Gree leaderboard identifiers (NSString) and the objects are GameCenter leaderboard categories (NSString).

NSString* const GreeSettingEnableLogging

Toggle logging.

GreePlatform logging is enabled by default. You can disable all logging by providing this setting with a value of [NSNumber numberWithBool:NO].

NSString* const GreeSettingWriteLogToFile

Toggle writing log to a file.

Logging to file is disabled by default. You can enable it with a value of [NSNumber numberWithBool:NO]. Logs will be written to Library/Logs/.

NSString* const GreeSettingLogLevel

Set the verbosity of logging output.

Setting to higher numbers corresponds to more logging output. Expected value is an [NSNumber numberWithInteger:GreeLogLevelPublic];

NSString* const GreeSettingUpdateBadgeValuesAfterRemoteNotification

Toggle whether the badge values are updated when a push notification is received.

Updating after a push notification is disabled by default. It can be enabled by setting [NSNumber numberWithBool:YES].

NSString* const GreeSettingUseWallet

If you are including libGreeWallet.a and intend to use GreeWallet functionality you must specify [NSNumber numberWithBool:YES].

By default GreeWallet is disabled.

NSString* const GreeSettingRemoveTokenWithReInstall

The platform AccessToken is persisted in the keychain by default (therefore it survives application uninstall.) You can remove the access token when reinstalling by enabling this setting with [NSNumber numberWithBool:YES].

NSString* const GreeSettingEnableGrade0

To allow Grade0 users to run an application, specify [NSNumber numberWithBool:YES].

disable is set by default.

NSString* const GreeSettingManuallyRotateGreePlatform

By default, GreePlatform will follow a normal UIKit-style rotation, changing its position to match the UIViewController in which it is maintained.

Changing this value to YES will allow the developer to manually set the rotation for GREE platform views. This is useful, for example, in OpenGL games where you may be locking your root view controller to portrait and performing rotations via OpenGL transforms.
