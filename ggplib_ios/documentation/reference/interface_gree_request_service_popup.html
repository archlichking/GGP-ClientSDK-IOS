Detailed Description

The GreeRequestPopup interface is used for interacting with the request service.

By using GreeRequestServicePopup, a player can request other players to do something (invite his friend to an event on the game, send a present to his friend, seek help from his friends, etc.) Developer can know how his request was proceeded.

For specifying behaviors of the popup dialog, this class provides following properties.

  • parameters : You can set parameters for the customization.
