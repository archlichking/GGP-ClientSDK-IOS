Property Documentation

- (GreeWidgetPosition) position [read, write, assign]

This widget's position can be adjusted to any of the 6 valid positions when expandable is set to YES, However, when expandable is set to NO, this widget only supports GreeWidgetPositionTopLeft and GreeWidgetPositionBottomLeft.

Note:
For non-expandable widget, it's position will be re-set to GreeWidgetPositionTopLeft if you pass in GreeWidgetPositionTopRight, and re-set to GreeWidgetPositionBottomLeft if you pass in GreeWidgetPositionBottomRight, GreeWidgetPositionMiddleRight, or GreeWidgetPositionMiddleLeft.
