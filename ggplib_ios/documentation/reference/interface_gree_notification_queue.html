Detailed Description

The GreeNotificationQueue interface manages a queue of GreeNotifications and their display.

It exposes methods to queue a notification as well as various properties relating to their presentation.

Note:
You can access the platform's notification queue via GreePlatform

Property Documentation

- (GreeNotificationDisplayPosition) displayPosition [read, write, assign]

The location on the screen where the notifications should be drawn.

Changing this value will draw the notification on the top or bottom of the screen. The drawing action takes into account the values of this property and interfaceOrientation, so that if the device is rotated, the top and bottom position will change to the top and bottom appropriate for the given orientation.

- (BOOL) notificationsEnabled [read, write, assign]

A flag which controls whether or not the notification is enabled.

If YES, notifications will be displayed. If NO, notifications will not be displayed. Any notifications received while the flag is NO will be discarded by the queue.
