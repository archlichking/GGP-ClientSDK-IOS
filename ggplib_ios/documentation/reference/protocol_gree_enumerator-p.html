Detailed Description

Generic protocol for enumerating collections of server-side resources.

Objects conforming to the GreeEnumerator protocol are created by the component class APIs to page over any (potentially) large set of data. In most cases you can expect id<GreeEnumerator> objects to be returned from the various API class methods beginning with load.

The typical API usage pattern looks something like this: 

  [GreeScore loadTopScoresForLeaderboard:leaderboard_id timePeriod:GreeScoreTimePeriodAlltime block:^(NSArray* scores, NSError* error) {
     // process top page of scores
  }];

with enumerators, however, you can retrieve more than the first page of data:

  id<GreeEnumerator> allScoreEnumerator = [GreeScore loadTopScoresForLeaderboard:@"" timePeriod:GreeScoreTimePeriodAlltime block:^(NSArray* scores, NSError* error) {
     // process top page of scores
  }];
  // some time later... after user presses "next", etc.
  [allScoreEnumerator loadNext:^(NSArray* scores, NSError* error) {
     // process second page of scores
  }];

When you have exhausted the data set your returned items will be nil.

Member Function Documentation

- (NSInteger) startIndex

Starting index for your next page request.

loadPrevious attempts to load the 'pageSize' items before 'startIndex' loadNext attempts to load the next 'pageSize' items after and including 'startIndex'

- (NSInteger) pageSize

How many entries, at maximum, each page will contain.

Note:
This is merely a suggestion; the server is free to cap this value if the page size is too large.
- (BOOL) canLoadPrevious

YES if there are items before the most recently loaded page, NO otherwise.

- (BOOL) canLoadNext

YES if there may be more data to load, NO if the data set is known to be exhaused.

- (void) loadNext: (GreeEnumeratorResponseBlock)  block

Loads the next page of data.

A page is defined as pageSize items starting with startIndex

Parameters:
blockThe GreeEnumeratorResponseBlock which will receive the items

- (void) setStartIndex: (NSInteger)  startIndex

Change the start index to a specified value.

Values < 1 will be set to 1.

Note:
Advanced API call, use with caution.
- (void) setPageSize: (NSInteger)  pageSize

Set a desired page size for your next load request.

It is not guaranteed that the server will accept this value. Once a page is loaded, this will be set to the page size from the server, changing this value after that point will interfere with paging backwards.

Note:
Advanced API call, use with caution.
- (void) setGuid: (NSString *)  guid

Sets the enumerator to use a specific user context.

Note:
Not all enumerators respect or support this notion.
Passing in nil will default to the current user.
Advanced API call, use with caution.
