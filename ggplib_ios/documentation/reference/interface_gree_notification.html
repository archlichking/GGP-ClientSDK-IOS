Detailed Description

The GreeNotification interface is the data model for a notification in the Gree Platform SDK.

A GreeNotification is used to convey information from the platform directly to the user. GreeNotification objects are generated by platform APIs or iOS remote/local notifications and displayed by being added to a GreeNotificationQueue.
