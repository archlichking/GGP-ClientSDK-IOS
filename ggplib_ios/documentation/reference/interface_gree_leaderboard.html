Detailed Description

The GreeLeaderboard interface encapsulates all of the Gree Platform's leaderboard functionality.

A leaderboard is a collection of scores in a single category of your application (i.e. highest score.) Leaderboards are defined on the GREE developer website. You can use the GreeLeaderboard interface to load leaderboard details. For details about how your users can populate your leaderboards with data, please see GreeScore.
