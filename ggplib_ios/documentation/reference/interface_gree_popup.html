Detailed Description

The GreePopup class provides an API to display pop-ups for various Gree Platform services.

GreePopup should not be used directly; rather, each service will define it's own subclass. Examples include GreeSharePopup, GreeRequestPopup, etc.

Display the receiver.

The popup will animate with a bounce animation from the center of the parentView. While the popup is displaying an opaque view will cover the parentView outside of the boundary of the popup.

- (void) dismiss

Dismiss the receiver.

The popup will animate out by shrinking toward the center of the parentView.
