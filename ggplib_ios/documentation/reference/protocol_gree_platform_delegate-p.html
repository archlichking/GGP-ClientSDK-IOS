Detailed Description

The GreePlatformDelegate protocol declares methods that should be implemented by the delegate of the GreePlatform object.

These methods are invoked when the Gree Platform needs to notify your application about important events that occur with regards to the Gree Platform and it's interaction with your application. Such events include user state changes (login, logout), modal views, etc.
